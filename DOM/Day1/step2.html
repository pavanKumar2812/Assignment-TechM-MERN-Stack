console.log() means a function that prints an array, object or message to the console most times as a string

console.dir() means a function that takes an object as its input and prints it out as a JSON-like tree


Console Object

It gives you access to the browsers console
The Console is part of the window object
It is supplied by Browser Object Model BOM

BOM-BROWSER OBJECT MODEL

BOM allows Javascript to talk to the browser about things other than the content of the page

  • Navigator - info about browser
  • Screen - current URL
  • Location - info about user's screen
  • History - info about user's history
There are no standards for the BOM
The whole HTML document is a property of the window object
