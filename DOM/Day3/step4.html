Types of Nodes


S.NO Constant value Description
1 ELEMENT_NODE 1 Most simple. just an element node like p or h1 tag
2 TEXT_NODE 3 The actual text
3 CDATA_SECTION_NODE 4 This is character data that is not parsed by the parser
4 PROCESSING_INSTRUCTION_NODE 7 Used with XML to embeded application-specific instructions
5 COMMENT_NODE 8 Pretty obvious it's a comment
6 DOCUMENT_NODE 9 Simple dimple This is our root document node
7 DOCUMENT_TYPE_NODE 10 This is so simple. Yep this is our DOCTYPE html declaration
8 DOCUMENT_FRAGMENT_NODE 11 A DocumentFragment node is lightweight document obvious sometimes you may want to extract a portion of the documnets tree and create a new fragment

There are only a handful of active (or relevant) node types

The 4 most common ones we will deal with are:

  1. Document Node
  2. Element Node
  3. Text Node
  4. Comment Node
