Traversing the DOM

How we move up and down the DOM
It begins with understanding how family structure works

Parent: A parent of any node is the node immediately above it

There are 2 main methods we can use to get the parent node

  1. parentNode
  2. parentElement

Child: The child of a node is the node immediately below it

Nodes beyond one level of nesting are called descendents


Sibling: A sibling of a node is any node in the same level

Sibling nodes don't have to be of the same type
